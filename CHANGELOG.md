# 更新日志

## v1.6.0  —  2026-07-06

### 装备栏（全新）
- “漫漫尘途”面板新增独立的【装备】栏，与【供应】**左右并排**：
  - **护甲**行：显示当前护甲 + `+HP` 加成（如“钢甲 (+35 hp)”）
  - **负重**行：显示当前载具 + 背包容量（如“行李架 (40)”）
  - **水**行：显示当前水容器 + 上限（如“水罐 (60)”）
  - **双手 / 大腿 / 小腿** 三类×2 槽，点方格弹出下拉选择已持有的武器
- 供应栏不再重复显示护甲/水（集中到装备栏）
- **自动联动**：当某件武器库存降到 0（回收/战斗消耗/交易），对应装备槽会自动卸下，不再残留无效引用

### 【回收】系统重做
- 直接在供应栏每一行右侧提供【**回收**】按钮（显式文字 + 边框），不再依赖游方铁匠事件触发
- **覆盖面扩大**：所有有制作/购买成本的物品均可回收，包括——
  - 制作类武器（骨枪/钢之刀/日轮刀/藤花铳……）
  - 锅炉制作物品（藤花油/丸药/霍火球/灵魂之钡/缚苦无/雷鸣铳……）
  - 商人售卖的弹药/消耗品（藤花弹/太阳结晶/药剂/锁镰/藤花榴弹……）
- 回收价 = 成本的 **30%**（向下取整）
- 回收后通知栏会【**发道什么材料 + 多少**】都写清楚（如“已回收 日轮刀 (木+150, 皮革+30, 钢+6)”）

### 弹药子行
- 供应栏武器行下方会自动列出弹药（如日轮铳 → — 太阳结晶），方便确认继续战斗能打多少发

### 名称调整（更贴合鬼灭之刃背景）
- “篷车” → 【**行李架**】
- “车队” → 【**辎重箱**】

### 其他
- 供应栏中武器与消耗品之间新增分隔线
- 下拉器选项现在带稀有度颜色，与背包内保持一致
- 夜间模式完整适配装备栏、分隔线、回收按钮

### 修复与体验优化 — 2026-07-07
- **中文显示修复**：修复了中文语言包损坏导致整个界面回退成英文的问题；并补全装备/回收相关翻译（回收、装备、负重、双手 / 大腿 / 小腿、卸下 等）
- **装备栏与供应栏布局**：加大两栏间距，并把【回收】按钮重新定位——
  - 不再遮挡携带数量的上/下调整箭头
  - 不再与装备栏方框重叠
- **回收机制优化**：
  - 回收**不再能回收「正在携带」的最后一件**：回收只消耗携带量之外的富余；当库存数量等于携带数量时，回收按钮自动隐藏
  - 当某物品的**携带数量减到 0 时，装备栏中对应的该物品会自动卸下**（例如取消携带日轮刀，装备栏里的日轮刀同步卸下）

## 2026-07-06

### 通知栏更醒目
左边通知会按内容自动着色，重要事件一眼可辨：
- **金色**：成就解锁
- **紫色**：永久增益
- **绿色**：获得奖励
- **青色**：远征结算
- **红色**：队员陨落 / 失败 / 报告
- **蓝色**：柱级 NPC 出场（胡蝶忍、竈门、鬼舞辻、煉獄、宇髓、时透、不死川、悲鸣屿、伊黑、甘露寺、富冈、各柱等）

夜间模式下颜色略微提亮，保持在深色背景下可读。刷新后历史消息也会按分类显示。

同时修复了通知加边框后右边一个字被截掉的显示问题。

### 自动伐木/查陷阱按钮改进
- **解锁提前**：以前要村民住满上限才解锁自动化，现在只要**造出第一间小屋就永久解锁**
- **首次解锁有明显提示**：紫色公告 + 按钮金色脉冲高亮，点一次高亮消失
- **不再消失**：以前按钮会随着人口波动反复出现/消失，现在解锁后永远保留在原位，样式跟旁边的伐木/查陷阱按钮统一
- 存档持久化，重启游戏后状态照常

### 出发面板重排
- **一键装满** 现在是标准按钮样式，跟"出发"完全一致
- 位置：**「一键装满」和「出发」并排显示**（左边一键装满，右边出发），不再挤在补给栏里

### 村落信息对齐修复
村落面板里建筑数字、村民分工数字现在整齐对齐到右边同一竖列，1 / 10 / 100 不再错位；也不会再因为中文名字过长把数字挤到下一行。

### 背包详情框修复
指向背包**最后一个物品**时，物品详情框会向上弹出，不再撑出多余的下拉条。

### 新增：游方铁匠（旅行商人）
- **每 10 分钟**会有一位游方铁匠停到营地，标题闪烁 + 紫色公告提示
- 商人会**按 30% 材料价回收多余的武器**，一键把用不上的旧武器换回材料
- 面板会自动列出你当前持有的每一件武器，显示回收后能拿回哪些材料 + 数量
- 数量归零后按钮自动禁用，点"目送他离开"结束

出现条件：你在营地 room、当前没有别的事件、地图已解锁、至少有 1 件可回收的武器。
