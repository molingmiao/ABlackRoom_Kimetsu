# 更新日志

## v1.6.0  —  2026-07-06

### 装备栏（全新）
- “漫漫尘途”面板新增独立的【装备】栏，与【供应】**左右并排**：
  - **护甲**行：显示当前护甲 + `+HP` 加成（如“钢甲 (+35 hp)”）
  - **负重**行：显示当前载具 + 背包容量（如“行李架 (40)”）
  - **水**行：显示当前水容器 + 上限（如“水罐 (60)”）
  - **双手 / 大腿 / 小腿** 三类×2 槽，点方格弹出下拉选择已持有的武器
- 供应栏不再重复显示护甲/水（集中到装备栏）
- **自动联动**：当某件武器库存降到 0（回收/战斗消耗/交易），对应装备槽会自动卸下，不再残留无效引用

### 【回收】系统重做
- 直接在供应栏每一行右侧提供【**回收**】按钮（显式文字 + 边框），不再依赖游方铁匠事件触发
- **覆盖面扩大**：所有有制作/购买成本的物品均可回收，包括——
  - 制作类武器（骨枪/钢之刀/日轮刀/藤花铳……）
  - 锅炉制作物品（藤花油/丸药/霍火球/灵魂之钡/缚苦无/雷鸣铳……）
  - 商人售卖的弹药/消耗品（藤花弹/太阳结晶/药剂/锁镰/藤花榴弹……）
- 回收价 = 成本的 **30%**（向下取整）
- 回收后通知栏会【**发道什么材料 + 多少**】都写清楚（如“已回收 日轮刀 (木+150, 皮革+30, 钢+6)”）

### 弹药子行
- 供应栏武器行下方会自动列出弹药（如日轮铳 → — 太阳结晶），方便确认继续战斗能打多少发

### 名称调整（更贴合鬼灭之刃背景）
- “篷车” → 【**行李架**】
- “车队” → 【**辎重箱**】

### 其他
- 供应栏中武器与消耗品之间新增分隔线
- 下拉器选项现在带稀有度颜色，与背包内保持一致
- 夜间模式完整适配装备栏、分隔线、回收按钮

### 修复与体验优化 — 2026-07-07
- **中文显示修复**：修复了中文语言包损坏导致整个界面回退成英文的问题；并补全装备/回收相关翻译（回收、装备、负重、双手 / 大腿 / 小腿、卸下 等）
- **装备栏与供应栏布局**：加大两栏间距，并把【回收】按钮重新定位——
  - 不再遮挡携带数量的上/下调整箭头
  - 不再与装备栏方框重叠
- **回收机制优化**：
  - 回收**不再能回收「正在携带」的最后一件**：回收只消耗携带量之外的富余；当库存数量等于携带数量时，回收按钮自动隐藏
  - 当某物品的**携带数量减到 0 时，装备栏中对应的该物品会自动卸下**（例如取消携带日轮刀，装备栏里的日轮刀同步卸下）
  - **回收最后一件后立即刷新**：修复了回收使某物品库存归零后、供应行仍残留、需要 Ctrl+F5 才消失的问题
- **地图（远征）界面精简**：进入地图后，左侧不再显示天赋 / 物品 / 装备栏，只保留地图（远征补给仍由地图自带的补给条显示）；安全归来或阵亡返回营地后自动恢复

### 无限城（Infinity Castle）大改 — 2026-07-07
- **主题正名为「坠落」**：进入过渡由原来的向上（升空感）改为**向下坠落**，与「一层层往下沉」的设定一致；入口按钮由错误的“进入地铁站”改为【**坠落**】
- **坠落前确认框**：点坠落先弹确认——「坠落至无限城？」，可选择【坠落】/【调整背包】（跳到漫漫尘途整理）/【等等】
- **背包制（与远征一致）**：无限城的整顿、战斗、商店统一使用**背包（Path.outfit）**；坠落即带走背包（从家里库存扣除），归来/阵亡自动结算归还
- **顶部随身背包展示**：无限城顶部像地图一样列出当前携带物资
- **随时离开按钮**：楼层右上角【离开无限城】，随时带着剩余背包返回
- **影中商人**：
  - 只出售无限城内**有用的回血品**（药剂 / 藤花精油），移除无用的火把、藤花符
  - **紫藤屋后援**：花费家里材料（木/鳞）买入当前背包
  - 每次进店各商品**随机限购 3~10 件**，售罄提示
  - 购买反馈明确：「购得 药剂（+1，共 N）；本店还剩 M 件」
  - 有几率出售【**神秘药水**】（赌博式，买即随机饮下）
- **药水系统（购买/开箱即饮，只影响下一场战斗）**：7 种药水分三类——
  - **增益**：力之药（增伤）/铁壁药（减伤）/活力药（开场满血）
  - **祸福相依**：狂战之酿（增伤但易受伤）/龟甲药汤（减伤但输出降）
  - **诅咒**：浊血瓶（受伤加重）/虚弱之瓶（输出降低）
  - 商人 70% 概率出售，宝箱 35% 概率含药水
- **未拾取掉落归仓**：战斗后你没拿走的掉落，由后援队伍直接送回紫藤庄园仓库
- **整顿栏**：按钮汉化并显示携带数量（如“吃熏肉（+8 生命） ×5”）
- **引路鸦计数**：改为固定宽度边框框，数字不再飘到右上角
- **最近击杀成就进度**：楼层状态栏显示最近的击杀里程碑（如“鬼杀新兵 38/50 只鬼”）
- **战斗界面修复**：
  - 修复无限城战斗中武器按钮**颜色丢失**（近战红/远程橙/眩晕蓝等类型色）
  - 战斗按钮**固定排序**：双手 → 副手 → 道具 → 药品，始终一致
  - 修复背包超重时掉落「一键带走」显示负数（如“带走 -102”）的问题

### 归阵符与日轮铳
- **归阵符（fleet beacon）新增战斗用途**：持有时战斗中出现【召集柱】按钮，消耗 1 张召集一位柱前来驰援、**瞬间回满血**（此前它几乎只用于加分）
- **日轮铳可制作/回收**：新增日轮铳制作配方（木 500 / 皮革 60 / 钢 20 / 硫磺 20，解锁条件同日轮刀），从此也能回收

### 无限城战力阶梯与 Rougelike 化 — 2026-07-07
- **敌人数值大幅提升**：初层普通鬼 **200+ HP**（原 8），层层递增；精英 ×1.8 HP / ×1.5 伤害；Boss 从 80/130/200 → **1200/2400/4500**
- **攻击频率翻倍**：敌人攻击间隔缩短一半（0.5~1.0s）
- **血鬼术系统**：精英/Boss 会释放血鬼术——
  - **出血**：命中后每秒扣血，持续 10 秒
  - **蓄力破盾**：5 秒预警，若有护盾则击破护盾，否则造成 2× 伤害
  - **狂乱连斩** / **血雾结界**：高频次或范围伤害
- **无惨 5 阶段战**：血量 60000（前中期血条几乎无变化），每 60 秒推进一个阶段——
  - 每阶段柱赶到：回满血 + 药剂+3 + 归阵符+1 + 削 10000 血
  - 标题栏实时显示【**距天明 X:XX**】倒计时
  - 5 分钟熬到天明即胜利
- **鬼杀天赋（本次 run 内 Rougelike）**：普通/精英战胜后 **3 选 1**，可加已有等级或选新天赋——
  - **硬体术**：+5 最大生命/级
  - **快刀术**：+2% 武器伤害/级
  - **铁壁**：+1% 减伤/级（上限 30%）
  - **吸血术**：+1% 吸血/级（水/炎/雷呼吸法各额外 +5%，总上限 25%）
  - **巧手**：+1% 命中/级
  - **风刃**：-1% 冷却/级（上限 50%）
  - 每项 1~20 级；返回入口后清空（每次坠落都是新 build）
- **围剿节点**：新节点【demon ambush】——3~5 只鬼连打，胜利后由后援队伍按你消耗量的 **50% 补回**（药/藤花精油/熏肉/太阳结晶/藤花弹）
- **太阳咒纹祭坛**：献 30% HP 换 1 级随机天赋
- **柱之邂逅**：随机遇到一位柱，免费获得对应天赋 1 级（水柱→快刀术、虫柱→巧手、炎柱→硬体术、音柱→风刃、风柱→吸血术、岩柱→铁壁）
- **顶部显示当前天赋列表**（如 “天赋：吸血术 Lv.3　快刀术 Lv.2”）

- **节点颜色分类**：

## 2026-07-06

### 通知栏更醒目
左边通知会按内容自动着色，重要事件一眼可辨：
- **金色**：成就解锁
- **紫色**：永久增益
- **绿色**：获得奖励
- **青色**：远征结算
- **红色**：队员陨落 / 失败 / 报告
- **蓝色**：柱级 NPC 出场（胡蝶忍、竈门、鬼舞辻、煉獄、宇髓、时透、不死川、悲鸣屿、伊黑、甘露寺、富冈、各柱等）

夜间模式下颜色略微提亮，保持在深色背景下可读。刷新后历史消息也会按分类显示。

同时修复了通知加边框后右边一个字被截掉的显示问题。

### 自动伐木/查陷阱按钮改进
- **解锁提前**：以前要村民住满上限才解锁自动化，现在只要**造出第一间小屋就永久解锁**
- **首次解锁有明显提示**：紫色公告 + 按钮金色脉冲高亮，点一次高亮消失
- **不再消失**：以前按钮会随着人口波动反复出现/消失，现在解锁后永远保留在原位，样式跟旁边的伐木/查陷阱按钮统一
- 存档持久化，重启游戏后状态照常

### 出发面板重排
- **一键装满** 现在是标准按钮样式，跟"出发"完全一致
- 位置：**「一键装满」和「出发」并排显示**（左边一键装满，右边出发），不再挤在补给栏里

### 村落信息对齐修复
村落面板里建筑数字、村民分工数字现在整齐对齐到右边同一竖列，1 / 10 / 100 不再错位；也不会再因为中文名字过长把数字挤到下一行。

### 背包详情框修复
指向背包**最后一个物品**时，物品详情框会向上弹出，不再撑出多余的下拉条。

### 新增：游方铁匠（旅行商人）
- **每 10 分钟**会有一位游方铁匠停到营地，标题闪烁 + 紫色公告提示
- 商人会**按 30% 材料价回收多余的武器**，一键把用不上的旧武器换回材料
- 面板会自动列出你当前持有的每一件武器，显示回收后能拿回哪些材料 + 数量
- 数量归零后按钮自动禁用，点"目送他离开"结束

出现条件：你在营地 room、当前没有别的事件、地图已解锁、至少有 1 件可回收的武器。
